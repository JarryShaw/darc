Data Models¶

The darc.model module contains all data models defined for the darc project, including RDS-based task queue and data submission.

  • Task Queues
    • Hostname Queue
    • Crawler Queue
    • Loader Queue
  • Submission Data Models
    • Hostname Records
    • URL Records
    • robots.txt Records
    • sitemap.xml Records
    • hosts.txt Records
    • Crawler Records
    • Loader Records
