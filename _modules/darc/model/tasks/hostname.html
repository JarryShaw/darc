"""Hostname Queue
--------------------

.. important::

   The hostname queue is a **set** named ``queue_hostname`` in
   a `Redis`_ based task queue.

   .. _Redis: https://redis.io

The :mod:`darc.model.tasks.hostname` model contains the data model
defined for the hostname queue.

"""
